macOS utility

Cmd+Tab for real windows.

ApSwitcher makes macOS switch across windows instead of only across apps. It is built for people who want the faster Windows-style flow without a bloated utility.

Get the DMG Open repository

What matters

  • Switches individual windows, not app buckets
  • Supports recent-window order and reverse cycling
  • Uses Accessibility for control and Screen Recording for thumbnails
Public build: v0.0.4

Why it feels different

No app stack guessing.

If you have two browser windows and one terminal, you see three targets. The selector stays faithful to what is actually open.

Install path

Simple and notarized.

Download the notarized DMG, move the app into Applications, approve Accessibility, and enable Screen Recording if you want previews.
