macOS utility

Cmd+Tab for real windows.

ApSwitcher makes macOS switch across windows instead of only across apps. It keeps the workflow closer to Windows while staying small, direct, and easy to install.

Download latest DMG View repository
  • Switches windows, not app groups
  • Cmd+Shift+Tab and arrow navigation supported
  • Notarized DMG releases on GitHub

What it actually looks like

The current app flow in use: centered switcher, per-window selection, and fast cycling between real windows.

Why it matters

Every window gets its own slot

Two browser windows and one terminal appear as three targets. The switcher stays tied to what is really open.

Install

Simple setup

Download the DMG, move the app into Applications, approve Accessibility, and enable Screen Recording if you want thumbnails.
