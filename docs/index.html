macOS utility

Window switching for macOS with Cmd+Tab.

ApSwitcher switches real windows, not just apps. It keeps the flow closer to Windows while staying small and practical on macOS.

v0.0.4

Switch windows, not apps

Every top-level window gets its own slot, so two browser windows and one terminal appear as three separate choices.

Built for fast back-and-forth

Recent-window ordering, Cmd+Shift+Tab, arrow navigation, and window previews make the selector usable in real work, not just as a toy.

Simple install path

Download the notarized DMG, move the app into Applications, grant Accessibility, and optionally enable Screen Recording for thumbnails.

Source

Open source and release-driven.

The repository includes the app source, release automation, and notarized DMG builds.
